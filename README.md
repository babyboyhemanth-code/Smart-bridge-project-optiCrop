# 🌱 OptiCrop
### Smart Crop Recommendation and Yield Prediction System

OptiCrop is an AI-powered web application that helps farmers make informed agricultural decisions by recommending the most suitable crops based on soil and environmental conditions. The system also predicts crop yield using Machine Learning, enabling efficient resource utilization and improved agricultural productivity.

---

## 📌 Features

- 🌾 Crop Recommendation based on soil nutrients and climate
- 📈 Crop Yield Prediction using Machine Learning
- 🌦 Weather-based agricultural insights
- 👨‍🌾 User-friendly web interface
- 📊 Fast and accurate predictions
- 💾 Secure data management

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/OptiCrop.git
```

### 2. Navigate to Project Folder

```bash
cd OptiCrop
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Browser

```
http://127.0.0.1:5000/
```
